# 一、值类型和引用类型
## 题目
```
1. typeof能判断哪些类型
undefined object number string boolean function

2. 何时使用===何时使用==
===为类型和值都得一样；==类型可不一样

3. 值类型和引用类型的区别
现象：原始值之间赋值，一个改变，另一个不变；引用值之间赋值，一个内部属性发生变化，另一个也跟着变；
本质：原始值的值数据结构为栈，赋值的是值，而引用值数据结构是堆，赋值后保存的内存地址；
为什么：出于性能的考虑，基于内存空间和cpu耗时；
```
